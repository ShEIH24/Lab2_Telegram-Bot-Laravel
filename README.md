# Telegram Shop Bot

Telegram-бот интернет-магазина на PHP с использованием библиотеки [irazasyed/telegram-bot-sdk](https://github.com/irazasyed/telegram-bot-sdk). Работает в режиме long polling. Подключается к MySQL-базе данных магазина.

---

## Возможности

### Для клиента
- **Каталог товаров** — просмотр по категориям, карточка товара с описанием, ценой и наличием
- **Поиск** — поиск по названию и описанию товара
- **Корзина** — добавление товаров, изменение состава, просмотр итоговой суммы
- **Оформление заказа** — выбор способа оплаты через кнопки (наличные, карта, СБП), подтверждение
- **Мои заказы** — список своих заказов со статусами и составом
- **FAQ** — ответы на вопросы о доставке, оплате, возврате, гарантии
- **Inline-режим** — поделиться карточкой товара в любом чате Telegram

### Для администратора
- Вход через команду `/admin` (только для указанных Telegram ID)
- Список всех заказов магазина с иконками статусов
- Поиск заказа по номеру или Telegram ID покупателя
- Просмотр детальной информации о заказе
- Смена статуса заказа: Подтверждён / Отправлен / Отменён
- Email-уведомление при каждом новом заказе

### Технические особенности
- При оформлении заказа остатки товаров списываются в одной SQL-транзакции с блокировкой строк (`FOR UPDATE`)
- Если товара не хватает — транзакция откатывается, корзина клиента не очищается
- Вся навигация по меню работает через редактирование одного сообщения (`editMessageText`), новые сообщения не добавляются
- Состояния пользователей (ожидание ввода) хранятся в JSON-файле
- Корзина каждого пользователя хранится в отдельном JSON-файле

## База данных

Бот работает с существующей БД магазина `magazin`. Используемые таблицы:

| Таблица | Что читает / пишет |
|---|---|
| `category_items` | читает список категорий |
| `items` | читает товары, уменьшает поле `count` при заказе |
| `bot_orders` | создаёт и читает заказы бота (своя таблица) |

Таблица `bot_orders` создаётся через `schema_orders.sql`:

```sql
CREATE TABLE bot_orders (
    id             BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    telegram_id    BIGINT NOT NULL,
    telegram_name  VARCHAR(200) NOT NULL DEFAULT '',
    items_json     TEXT NOT NULL,
    total          DECIMAL(10,2) NOT NULL,
    payment_method VARCHAR(50) NOT NULL,
    status         VARCHAR(30) NOT NULL DEFAULT 'new',
    created_at     TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
